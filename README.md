# csit321
-----------------------------
- install node.js

- load the library
  ```
  cd backend/package.json
  npm install
  ```
  
  ```
  cd webapp/package.json
  npm install
  ```

- To run backend server
  ```
  cd backend\src
  node main.js
  ```

- To run React server
  ```
  cd webapp
  npm start
  ```

  view the web at http://localhost:3000
